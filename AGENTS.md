# AGENTS

## 목적
- iPhone 현장 사용을 전제로 한 감귤 생육조사 앱을 Expo React Native 기반으로 유지보수한다.

## 원칙
- 모바일, 특히 iPhone 실기기 동작을 우선한다.
- 오프라인 우선으로 설계하고 로컬 SQLite 저장을 기본 경로로 유지한다.
- 음성 입력 품질과 복구 가능성을 UI 화려함보다 우선한다.
- 큰 리팩터링보다 작은 검증 가능한 변경을 우선한다.
- 모든 의미 있는 변경 뒤에 `lint`, `test`, `typecheck`, 가능한 빌드 검증을 수행한다.

## 응답 형식
- 현재 상태
- 계획
- 수정 파일
- 검증
- 남은 이슈

## 구현 원칙
- 서비스 계층, 상태관리, 화면, 상수를 분리한다.
- 요구사항에 없는 유료 서비스는 도입하지 않는다.
- 실패, 타임아웃, 권한 거부, 네트워크 불가, 모델 미다운로드 상태를 반드시 드러낸다.
